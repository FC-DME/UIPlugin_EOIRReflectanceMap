About the EOIR Reflectance Map Generator plugin

The EOIR Reflectance Map Generator plugin allows users to generate a relflectance map to use with Ansys' Systems Tool Kit EOIR capabilities.

STK 12.5 introduced the capability to create custom reflectance, emissivity, temperature, and radiance maps to add fidelity to EOIR analyses.

This plugin allows uers to automate the creation of custom reflectance maps. It takes a screenshot of the open 2D window, processes the image and re-exports it as a .csv file readable by STK's EOIR module. Users can select whether they want the created map to be added to the scenario's EOIR configuration.

How to Use the Plugin

Users first need to set up the 2D window to ensure a good quality image is used. It is recommended to hide all the scenario objects and latitude and longitude lines to ensure they do not interfere with the map created. Users can use the radio buttons to hide these elements then show them again once the map has been generated.

In the Image Setup section, users can choose how to name the map and browse to the location they want to save the image and .csv file generated.

Finally, users can use the Add reflectance map to the EOIR configuration for this scenario checkbox to select whether they want the generated map to be added to the EOIR configuration automatically or not. If the checkbox is not selected, users can find the minimum and maximum latitude and longitudes required to add the map manually to the EOIR configuration in the STK Message Viewer.
